# Práctica Git

Este repositorio es un proyecto de práctica para aprender Git y GitHub desde cero.

## Objetivo

El objetivo de este proyecto es practicar los comandos básicos de Git:

- git init
- git status
- git add
- git commit
- git log
- git diff
- git restore
- .gitignore

## Tecnologías utilizadas

- Python
- Git
- Visual Studio Code

## Estado del proyecto

Proyecto en fase de aprendizaje.

## Rutina de trabajo

Antes de empezar a trabajar en un proyecto con Git, revisaré el estado con `git status` y actualizaré con `git pull`.

Al terminar, guardaré los cambios con `git add`, `git commit` y `git push`.
