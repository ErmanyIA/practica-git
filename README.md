# Práctica Git

Este repositorio es un proyecto de práctica para aprender Git y GitHub desde cero.

## Objetivo

El objetivo de este proyecto es practicar los comandos básicos de Git:

- git init
- git status
- git add
- git commit
- git log
- git diff
- git restore
- .gitignore

## Tecnologías utilizadas

- Python
- Git
- Visual Studio Code

## Estado del proyecto

Proyecto en fase de aprendizaje.
